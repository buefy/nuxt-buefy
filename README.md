## Install

```bash
npm i nuxt-buefy
# or
yarn add nuxt-buefy
```

## Usage

```js
{
  modules: [
    // Simple usage
    'nuxt-buefy',

    ['nuxt-buefy', { /* buefy options */ }]
 ]
}
```

or 

```js
{
  modules: [
    // Simple usage
    'nuxt-buefy',
 ],
 buefy: { /* buefy options */ }
}
```

## TypeScript

Add `buefy/types` to `compilerOptions.types` of `tsconfig.json`.
